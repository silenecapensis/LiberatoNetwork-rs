# RFC #1: Liberato-Digital-Asset-Standard (LDAS-1)

## Purpose

The purpose of this RFC is to create a standard for Digital Assets in Liberato. There are multiple ways it can be defined. Digital Assets serve an important purpose in Liberato and are comparable to NFTs.

## LDAS-1: Standard For Digital Assets

**Hash:** BLAKE2B(48)
**Encoding:** Hexadecimal
